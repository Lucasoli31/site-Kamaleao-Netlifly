# 🍹 Kamaleão Drinkeria – Cardápio Digital Interativo

Aplicação web desenvolvida para apresentação de cardápio digital com foco em experiência do usuário, navegação dinâmica e captação de clientes, utilizando recursos de cloud para armazenamento e gerenciamento de dados.

---

## 🚀 Sobre o Projeto

O projeto Kamaleão Drinkeria foi criado com o objetivo de modernizar a apresentação do cardápio de um bar, oferecendo uma interface interativa e responsiva, além de um sistema de cadastro de clientes integrado a uma solução em nuvem.

A aplicação permite que usuários naveguem por categorias de drinks, visualizem detalhes dos produtos e realizem cadastro para futuras ações de relacionamento e marketing.

---

## ☁️ Arquitetura Cloud

A aplicação utiliza serviços em nuvem para garantir escalabilidade, disponibilidade e persistência de dados:

* **Firebase (Google Cloud Platform)** como Backend-as-a-Service (BaaS)
* **Cloud Firestore (NoSQL)** para armazenamento de dados dos clientes
* Regras de segurança configuradas para controle de acesso (leitura e escrita)
* Armazenamento em tempo real e altamente disponível
